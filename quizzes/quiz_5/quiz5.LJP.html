(d) Flag subset of data and plot

  • Create a boolean colum (True/False) "bool_cond" which checks whether
    "growth_euro" had a value greater than 2 or less than -2
  • Generate a line plot with the "growth_euro" series, label the axes and the title
  • Use the command plt.fill_between() as in Lecture 21 to create a shaded region
    for time periods where "bool_cond" is true.
In[]:
# Write your own code

currencies['bool_cond']= ((currencies['growth_euro']>2) | 
                          (currencies['growth_euro']< -2))

plt.plot('date','growth_euro',data=currencies)
plt.xlabel('Date')
plt.ylabel('Percent Daily Growth')
plt.title('Daily Growth of Euro Outside of Range')

vec_y=currencies['growth_euro']
plt.fill_between(x=currencies['date'],
                 y1=vec_y.min(),
                 y2=vec_y.max(),
                 where = currencies['bool_cond'],
                 alpha =0.2, color = 'red')

  • Follow the instructions in Lecture 20 to compute an aggregate dataset
    by month which computes the standard deviation of "eur_to_usd"
  • Display the dataset
In[]:
# Write your own code

month_config = pd.Grouper(key='date',freq='m')

monthly_data = (currencies
                .groupby(month_config)
                .agg(std = ('eur_to_usd','std'))
                .reset_index())
